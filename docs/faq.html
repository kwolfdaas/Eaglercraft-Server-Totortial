❓ Frequently Asked Questions

Quick answers to common questions about Eaglercraft.

Legal & Copyright

❓ Is Eaglercraft legal?

Short answer: It's a gray area. Microsoft considers Eaglercraft copyright infringement.

Private use (Low Risk): Playing alone or with friends on your own server is generally safe.

Public hosting (Higher Risk): Running a large public server could draw Microsoft's attention and DMCA takedowns.

Bottom line: Most people play safely with zero issues. Just be aware of the risk.

❓ Will Microsoft shut down Eaglercraft?

Possibly. Microsoft has sent DMCA notices to public servers before, but hasn't pursued individual players.

❓ Can I get in trouble for playing Eaglercraft at school?

You could get in trouble with the school, but not legally. Schools block it for bandwidth, not legal reasons.

Gameplay & Features

❓ Can I use mods?

Traditional Forge mods: No, Eaglercraft doesn't support them.

Featured mods: Some clients include built-in features (like keystrokes display or FOV changes).

❓ Does Eaglercraft work on Chromebooks?

Yes! Chromebooks work great with Eaglercraft:

  • ✅ No installation needed
  • ✅ 40-60 FPS typical
  • ✅ Works in Guest Mode
  • ❌ May be blocked by school filters

❓ Can I play with my friends?

Yes, two ways:

  • Shared Worlds: Peer-to-peer worlds (direct connection, no server needed)
  • Multiplayer Server: Host a server (Codespaces, Sealos, or VPS)

❓ Do I need a Minecraft account?

No. Eaglercraft runs in offline mode. You can play with any username.

❓ Can I use my real Minecraft skins?

Partially. Eaglercraft can't access Mojang's servers, but you can upload PNG skin files manually.

❓ Will my world get deleted?

Your world is stored in browser local storage. It persists unless you:

  • ❌ Clear your browser cache/cookies
  • ❌ Uninstall the browser
  • ❌ Switch browsers or devices

Pro tip: Export your world regularly as a backup.

Hosting & Servers

❓ Can I run a free server?

Yes! Multiple options:

  • ✅ GitHub Codespaces (120 hours/month free)
  • ✅ Sealos free tier
  • ✅ Run locally on your computer

❓ How many players can join?

Depends on your server:

  • Small server (Codespaces): 10-20 players
  • Medium server (Sealos free): 30-50 players
  • Large server (VPS): 100+ players

❓ Can I add plugins?

Yes, with Spigot or Paper. Some popular plugins:

  • Essentials (teleport, spawn points)
  • LiteBans (moderation)
  • AntiCheat (prevent cheating)

❓ Can I make money from a server?

No. Running a commercial/paid Eaglercraft server violates Minecraft's EULA.

Performance & Technical

❓ Why is my FPS low?

Most common causes:

  • Too many browser tabs open
  • Shaders enabled on weak hardware
  • Render distance set too high
  • Browser doesn't support WASM-GC

Solution: See our FPS guide

❓ Does Eaglercraft use a lot of data?

Moderate usage: ~50-100 MB per hour of gameplay. Similar to YouTube.

❓ Can I play offline?

Mostly.Singleplayer works offline (once loaded). Multiplayer requires internet for login.

❓ What browsers work best?

  • ✅ Chrome 90+ (best performance)
  • ✅ Firefox 89+ (good performance)
  • ✅ Edge 90+ (very good)
  • ⚠️ Safari (works but slower)
  • ❌ Very old browsers won't work

Safety & Security

❓ Is Eaglercraft safe?

If played from official sources: Yes, it's safe.

Unsafe sources: Random "unblocked games" websites sometimes serve malicious versions with:

  • 💀 Malware
  • 💀 Crypto miners
  • 💀 Tracking software

Only download from official GitHub or trusted launchers.

❓ Does voice chat expose my IP?

Yes. WebRTC voice chat can reveal your IP to other players.

Solution: Use a VPN when voice chatting.

❓ Can I get hacked?

Unlikely if you:

  • ✅ Download from official sources
  • ✅ Don't run untrusted code
  • ✅ Use a modern browser with updates
