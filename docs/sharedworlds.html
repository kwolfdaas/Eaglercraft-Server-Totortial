🌍 Shared Worlds (Peer-to-Peer)

Play with friends without hosting a server.

Shared Worlds = No server needed. Instead, you and your friends connect directly (peer-to-peer). Easy, no cost, no setup required.

Shared Worlds vs. Servers

Feature Shared World Server
Setup Instant (in-game) 5-10 minutes
Cost Free Free-$20/month
Hosting One player's computer Dedicated server
Player Limit 2-8 players 10-100+ players
Uptime Only if host is on 24/7
World Persistence On host's computer Server storage

How Shared Worlds Work

When you create a Shared World:

  1. Your Eaglercraft client becomes the "host"
  2. Other players join by entering your invite code
  3. Data syncs peer-to-peer (direct connection)
  4. No external server involved
  5. World data stored on your computer

Advantages:

  • ✅ Instant setup (no Docker, no VPS)
  • ✅ Complete privacy
  • ✅ Full control
  • ✅ Zero cost

Disadvantages:

  • ❌ Host must be online
  • ❌ Limited to small groups
  • ❌ Slower if host has poor internet
  • ❌ No 24/7 availability

Creating a Shared World

Step 1: Open Eaglercraft

Launch your Eaglercraft client.

Step 2: Go to Singleplayer

  • Click "Singleplayer"
  • Create a new world or select existing world

    • Step 3: Open to LAN

    While in a world:

  • Press ESC
  • Click "Open to LAN"
  • Select gamemode (Survival, Creative)
  • Toggle "Allow Cheats" if desired
  • Click "Start LAN World"

    • Step 4: Share the Code

    Eaglercraft generates an invite code. Share with friends:

    • 📱 Text message
    • 💬 Discord
    • 📧 Email
    • ✏️ Screenshot

    Joining a Shared World

    Step 1: Get the Invite Code

    The host shares: eaglercraft:1.8.8-dc08bc45f (example)

    Step 2: Open Multiplayer

  • Click "Multiplayer"
  • Click "Direct Connect"

    • Step 3: Paste the Code

  • Paste the invite code in the address bar
  • Click "Join Server"
  • Wait to load the world

    • Step 4: Play!

    You're now connected to the host's world. Build, explore, survive!

    Shared World Management

    How Many Players Can Join?

    • 2-4 players: Smooth experience
    • 5-8 players: Playable but may lag
    • 8+ players: Noticeable lag or disconnects

    Is My World Safe?

    Privacy: Your world data stays on your computer (not uploaded anywhere).

    Griefing: Players can build and destroy. No built-in protection unless you:

    • Use Creative Mode (can't destroy)
    • Manage players manually
    • Keep Shared Worlds small/trusted groups

    World Backup

    Shared Worlds save to your Eaglercraft folder. Back it up regularly:

    • Find: Browser > AppData > Eaglercraft > Shared Worlds
    • Copy folder to external drive
    • Restore if needed

    Troubleshooting Shared Worlds

    ❌ Problem: "Can't Connect to Server"

    • Host must be in the world (not paused)
    • Host's internet connection must be stable
    • Code might have changed
    • Try getting a new code from host

    ❌ Problem: Lag or Disconnections

    • Too many players (reduce to 4-5)
    • Host has poor internet
    • Try moving closer to wireless router
    • Host should reduce other internet usage

    ❌ Problem: World Data Lost

    • Host cleared browser cache
    • Host uninstalled browser
    • Data might be in browser backup
    • Check Chrome/Firefox profile folders

    Best Practices

    • ✅ Regular backups (monthly)
    • ✅ Keep group small (4-5 players max)
    • ✅ Trust your players (no griefing)
    • ✅ Communicate on Discord/chat
    • ✅ Use Creative Mode for building projects
    • ✅ Host should have stable internet
    • ❌ Don't leave host AFK too long (memory leak)

    When to Use Shared Worlds vs. Servers

    Use Shared World if:

    • 🎯 Just you + a few friends
    • 🎯 Want instant setup (0 minutes)
    • 🎯 Want zero cost
    • 🎯 Want complete privacy
    • 🎯 Playing occasional sessions

    Use Server if:

    • 🎯 Large player group (10+ people)
    • 🎯 Need 24/7 uptime
    • 🎯 Want permanent world (always on)
    • 🎯 Want to add plugins/mods
    • 🎯 Want to accept random players
