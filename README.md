### To Enable GitHub Pages (IMPORTANT!)

Since this is your first setup, you need to **enable GitHub Pages** in your repository settings:

### Step-by-Step GitHub Pages Setup:

1. **Go to your repository**: `https://github.com/kwolfdaas/Eaglercraft-Server-Totortial`
2. **Click Settings** (top right)
3. **Scroll down to "Pages"** (left sidebar)
4. **Under "Build and Deployment":**
   - **Source**: Select `Deploy from a branch`
   - **Branch**: Select `main`
   - **Folder**: Select `/docs`
5. **Click Save**

### ✅ After You Enable It:

Your site will be live at:
```
https://kwolfdaas.github.io/Eaglercraft-Server-Totortial/
```

It may take **1-5 minutes** for GitHub to build and deploy. If it's not working:
1. Wait a few more minutes
2. Do a hard refresh in browser: **Ctrl+Shift+R** (or **Cmd+Shift+R** on Mac)
3. Check the "Pages" section in Settings again — it should show a green checkmark
